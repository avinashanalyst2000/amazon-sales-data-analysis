# 📊 Amazon Sales Performance Analytics Dashboard

## 📝 Project Overview
This project presents an interactive Excel analytics solution designed to evaluate **3,000+ Amazon sales transactions**. The objective is to identify top-performing product categories, track regional profitability, monitor order delivery times, and provide actionable business insights for supply chain management.

---

## 🔑 Key Insights Identified
* **Top Revenue Driver:** The *Technology* category contributed to approximately 40% of total sales.
* **Profitability Champion:** While *Office Supplies* had lower transaction volumes, it maintained the highest consistent net profit margins.
* **Logistics Bottleneck:** The *West Region* experienced the longest average order delivery times (4.5 days), indicating a need for warehouse optimization.

---

## 🛠️ Advanced Excel Skills Applied
* **Data Dictionary Implementation:** Structured column definitions for seamless onboarding.
* **Dynamic KPI Blocks:** High-level metrics showing Total Sales, Net Profit, and Item Count instantly.
* **Interactive Slicers:** Connected multi-pivot visuals allowing managers to filter by *Year*, *Region*, and *Product Category* with one click.
* **Automation:** Embedded custom VBA routines to standardize gridline views and layout auto-fitting.

---

## ⚡ Instructions to Run
1. Download the `Amazon sales.xlsm` file from this repository.
2. Open it in Microsoft Excel (Desktop application recommended).
3. **Enable Macros** when prompted to allow automated UI optimization scripts to execute.
4. Navigate to the `Dashboard` worksheet and use the visual slicers to explore the data dynamically.
